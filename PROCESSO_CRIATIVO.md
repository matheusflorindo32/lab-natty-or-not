# PROCESSO_CRIATIVO.md — Processo criativo do projeto

Este documento detalha o fluxo de criação do projeto **IA aplicada à criação de conteúdo educacional para concursos**, desenvolvido para o Lab DIO **Natty or Not**.

## 1. Intenção do projeto

A intenção foi criar um material educacional com apoio de IA Generativa, mantendo clareza, utilidade e naturalidade.

Pergunta central:

> Como usar IA para criar conteúdo educacional sem deixar o resultado genérico ou artificial?

## 2. Público-alvo

O conteúdo foi pensado para estudantes de concursos públicos que precisam de materiais objetivos, organizados e fáceis de entender.

## 3. Fluxo de produção

| Etapa | Descrição | Resultado |
|---|---|---|
| Tema | Escolha do foco educacional | Conteúdo para concursos |
| Ideação | Uso de IA para organizar possibilidades | Estrutura inicial |
| Texto | Refinamento da mensagem | Conteúdo claro e direto |
| Visual | Organização gráfica do material | Peça educacional digital |
| Revisão | Ajustes de naturalidade e utilidade | Versão final mais humana |
| GitHub | Documentação da entrega | Repositório público |

## 4. Decisões de criação

- Priorizar linguagem simples.
- Evitar promessas exageradas.
- Evitar excesso de elementos visuais.
- Mostrar o uso da IA como apoio, não como substituição da curadoria humana.
- Organizar o repositório para avaliação e portfólio.

## 5. Resultado

O resultado foi uma entrega educacional simples, visual, documentada e adequada ao desafio proposto pela DIO.
